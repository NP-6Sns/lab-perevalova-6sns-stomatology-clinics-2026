Краткая методология подсчета
  • В выборку вошли только московские клиники.
  • Учтены клиники с не более чем тремя отделениями.
  • ТОП по выручке строится по ИНН: одна организация — одна строка.
  • Если у одного юридического лица несколько филиалов, выручка считается по юридическому лицу целиком.
  • Рейтинги и цены показываются по конкретным филиалам, поэтому одна сеть может встречаться несколько раз.
  • Цены для вкладки услуг: официальный сайт + агрегаторные источники (2ГИС/Яндекс) с фильтром аномально низких значений.
  • При равных значениях порядок определяется по отзывам и рангу в базе.
  • Актуальность данных: апрель–май 2026.
